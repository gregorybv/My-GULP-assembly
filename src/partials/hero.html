Why did I make this assembly?

Gulp-build is a powerful tool for automating the web application development process. I decided to make my own build to streamline the workflow and speed up web application development.
When creating my build, I thought about the fact that every developer has different preferences and requirements to the development process. So in my build, I made sure to allow for customization to meet specific needs.
In my build, I've included features such as image compression and optimization, automatic page refresh when code changes, CSS preprocessor compilation, and more.
